Welcome to the SilverGarden Programming Language!

My Object-Oriented Programming Language, that is designed to be
transpiled to C++, C and D!, the original implementation (current implementation)
is fully written in D. also, you can check up the source code at my GitHub page.

"Hello, World!" Example

a basic: "Hello, World!", unlike java you dont need to do it inside a class.
your main method does not require a strict name!, you only need the @Entry Annotation
and an argument which can be none or String[] args.


namespace "Silver.Examples"

import "Silver.Core"

public @Entry Main() {
	Console.Println("Hello, World!");
}
